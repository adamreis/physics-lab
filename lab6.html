Lab 6: Polarization and Interference¶

Introduction:¶

In this experiment we examined the effects of polarization on light intensity and the effects of interference from passing light through double and single slits on intensity. We found that the observed relative intensities of polarized light at different angles was able to explain 99.6% of the expected values. We expect it to explain 100% of the expected values, so 99.6% is close. Passing light through the double slit, we observed the wavelength of the He-Ne laser to be 797.72nm ± __ nm – since He-Ne lasers have a wavelength of 632.8nm this value is not within the range, and is therefore unexpected....

Procedure:¶

Polarization: First, we shine an incandescent light source unfiltered towards the fiber-optic cable hooked up to the photometer to obtain the maximum intensity of the light. We then place one polarizing lens in between, and then a second and record the intensities. Next we change the polarizing angle in intervals of 10 degrees and record the intensities.

Double Slit Diffraction Pattern: We shine the He-Ne laser through slit-set C and set it up such that the resulting interference pattern shines directly onto the red filter placed before the fiber-optic cable hooked up to the photometer on a linear translator. Starting at the center (.025m mark) we record the distances Xm and intensities where we observe a local maximum in the intensity reading on the photometer. We obtained 7 of these points. We also record the distance D from the slits to the fiber optic cable and the slit separation d.

Single Slit Envelope: We use the same setup as the Double Slit Diffraction Pattern, but we move the translator and fiber-optic cable closer, but not more than half as close as before, to the laser. We record this new distance D as well as the slit width a written on the component slide. We then traverse through the positions along the translator locating as many minima as we can find. We record these distances Xn and intensities – we found 15 of these minima.

import numpy as np

def lsqfity(X, Y):
    """
    Copied from http://stackoverflow.com/questions/27764220/retrieve-the-standard-deviation-of-the-y-intercept
    Calculate a "MODEL-1" least squares fit.

    The line is fit by MINIMIZING the residuals in Y only.

    The equation of the line is:     Y = my * X + by.

    Equations are from Bevington & Robinson (1992)
    Data Reduction and Error Analysis for the Physical Sciences, 2nd Ed."
    pp: 104, 108-109, 199.

    Data are input and output as follows:

    my, by, ry, smy, sby = lsqfity(X,Y)
    X     =    x data (vector)
    Y     =    y data (vector)
    my    =    slope
    by    =    y-intercept
    ry    =    correlation coefficient
    smy   =    standard deviation of the slope
    sby   =    standard deviation of the y-intercept

    """

    X, Y = map(np.asanyarray, (X, Y))

    # Determine the size of the vector.
    n = len(X)

    # Calculate the sums.

    Sx = np.sum(X)
    Sy = np.sum(Y)
    Sx2 = np.sum(X ** 2)
    Sxy = np.sum(X * Y)
    Sy2 = np.sum(Y ** 2)

    # Calculate re-used expressions.
    num = n * Sxy - Sx * Sy
    den = n * Sx2 - Sx ** 2

    # Calculate my, by, ry, s2, smy and sby.
    my = num / den
    by = (Sx2 * Sy - Sx * Sxy) / den
    ry = num / (np.sqrt(den) * np.sqrt(n * Sy2 - Sy ** 2))

    diff = Y - by - my * X

    s2 = np.sum(diff * diff) / (n - 2)
    smy = np.sqrt(n * s2 / den)
    sby = np.sqrt(Sx2 * s2 / den)

    return my, by, ry, smy, sby

Discussion:¶

Polarization: When we put on one polarizer, the intensity drops from 14 to 6.6, which is pretty close to the expected drop of 1/2. Since we had calibrated the photometer to measure 0 with the light turned off, we saw an intensity of 0 when the polarizers were orthogonal to one another, which is what we expect if there was no noise since there's no parallel component to the light that would be able to pass through the second filter (cos^2(90) == 0). If the background noise is constant than we can nullify it by properly calibrating the photometer as we did, which would eliminate the noise. Although we had a value of (0,0) on the graph, giving us an intercept of 0, the linear fit gives us an intercept of .027. This is close to 0, but signifies the 'noise level' present. If we had performed this experiment with the laser instead of the incandescent bulb, our results would be different since the light from a laser is already polarized. Thus attempting to pass it through the polarizers at any angle not aligned with the polarization would give us an intensity of 0 since no other component of the laser light would be at an angle other than its polarization angle. We see our slope is very close to 1 (1.011) and R^2 = .996, so the results are very good. We could, of course, have improved these readings if we used a digital photometer instead of analogue.

Double Slit Diffraction Pattern: We compare our observed wavelength of 797.72nm +/- 75.51nm to the expected value of 632.8nm. We see that it is not within the range and this must be due to some systematic error in our procedure. Looking at the graph we can immediately see that this is clearly not a linear set of data, closer to cubic. This is because the maxima were so close together, the linear translator so difficult to adjust systematically at those distances, and the sensitivity and reactivity of the photometer made it almost impossible to observe all the maxima in order. If the photometer was digital and reacted almost instantly to light striking it then we should be able to more easily find maximas, and if we were able to adjust the linear translator electronically we could feasibly write a program that would be much more consistent in locating maxima by looping through the translator in small sized chunks and then ‘zeroing in’ on maximums when it observes (current_intensity - prev_intensity)*(prev_prev_intensity – prev_intensity) < 0. This would be much more accurate than having humans guessing at maximas, which severely limits the precision of our measurements. If the slits were further from the laser then feasibly the particles from the air would reflect more of the laser light and thus lower the intensity, which could make it even more difficult to detect maxima of the fringes. However this would likely be a small amount, and since the laser light is a focused beam and if we assume that the amount of reflective particles in the air is roughly constant then we should see negligible difference. Changing the slit-to-fiber spacing would spread out the diffraction pattern so we would see less maxima in the 5cm range allowed by the linear translator and more space between maxima.

Single Slit Envelope: We found an estimated slit-width a of .0587mm +/- 0.0056mm which is pretty close to the actual value of .08mm given that our wavelength was off by ~160nm and this used that wavelength in its calculation (reflected in the error propagation). Taking the envelope of the double-slit intensity profile yields the intensity profile of light if you had passed it through a single slit. Thus taking the envelope again of the intensity profile of light after passing it through a single slit would be smaller relative to this as intensity tapers off exponentially in distance from center. We could improve these measurements in the same way we could have improved the double slit interference pattern portion of the experiment.

Conclusion:¶

We found that our measurements of relative intensity matched the expected theoretical amount extremely closely, at 99.6% explanation of the expected value. We found a wavelength of the He-Ne laser of 797.72nm +/- 75.51nm which was not within the range of the expected wavelength of 632.8nm. We also calculated a slit width of .0587mm +/- 0.0056mm which was out of the range of the actual value of .08mm. We can improve the results of these measurements dramatically if we can get a digital photometer coupled with a electronically controlled linear translator so that we can write a program to locate the minima, maxima and intensities at these locations within a set epsilon precision level.
